# 🐞 Border size increases each time the “Apply” button is clicked

## Environment
**Test Device:** Lenovo 10.0.26100 Build 26100
**Operating System:** Windows 11 Pro 24H2
**Browser:** Google Chrome - Version 140.0.7339.210
**Reproducibility Rate:** 100%
**Severity:** Major
**Priority:** High

## Steps to Reproduce:
1. Open the website: https://www.example.com/
2. Click the “Apply” button multiple times

## Actual Result:
Each time the user clicks the “Apply” button, the border around the button increases in size.

## Expected Result:
A red border appears around the “Apply” button after clicking it, but its size should remain unchanged regardless of how many times it is clicked.
