# 🐞 Live user crash when tapping on a user currently in Live

## Environment:
**Test Device:** iPhone 11

**Operating System:** iOS 26.1

**Application Version:** Version 1.0.0 (1.0.48)

**Reproducibility Rate:** 100%

## Pre-Conditions:
• The user is authenticated

## Steps to Reproduce:
1. Open the mobile application: Rebut
2. Tap on any Live session

## Actual Result:
The application crashes when the user taps on the name and surname of a user who is currently in a Live session.

## Expected Result:
The application should redirect the user to the selected user’s profile page.
