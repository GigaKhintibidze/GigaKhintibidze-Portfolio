# 🐞 Incorrect email and missing username in Edit Profile after Apple login

## Environment
**Test Device:** iPhone 11

**Operating System:** iOS 26.1

**Application Version:** Version 1.0.0 (1.0.56)

**Reproducibility Rate:** 100%

## Pre-Conditions:
• The user has a saved Apple account previously used for authentication

## Steps to Reproduce:
1. On the Rebut login page, tap “Continue with Apple”
2. Tap “Continue”
3. Tap “Sign in with Passcode”
4. Enter a valid passcode
5.Tap the profile icon
6. Tap “Edit Profile”

## Actual Result:
In the Edit Profile section, the username is not displayed and an incorrect email address is shown in the email field.

## Expected Result:
In the Edit Profile section, the username field should display the name associated with the Apple account, and the email field should show the correct Apple account email address.
