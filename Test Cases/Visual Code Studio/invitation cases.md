# N1
## Tittle: Invitation Creation

## Description: 
Creating an invitation by the user

## Precondition:

• The user is authorized

• The user is on the templates page

## Postcondition:
• The user creates an invitation

## Steps to Reproduce:

1. Open the website: https://invu.ge/
2. Click on the first template labeled: "simple wedding"
3. Fill in at least the required fields on the specific page
4. Click the button: "მოსაწვევის შექმნა"

## Expected Result:

• The user successfully creates an invitation.

# N2
## Tittle: Invitation Creation Without Required Fields

## Description: 
Creating an invitation by the user without filling in the required fields

## Precondition:

• The user is authorized

• The user is on the templates page

## Postcondition:
• The user cannot create an invitation

## Steps to Reproduce:
1. Open the website: https://invu.ge/
2. Click on the first template labeled: "simple wedding"
3. Do not fill in the required fields on the specific page
4. Click the button: "მოსაწვევის შექმნა"

## Expected Result:

• The system does not allow the user to click the invitation creation button until the required fields are filled in.
