# Saving Address Details on the Checkout Page

## Description:
Entering and saving the user’s address details on the Checkout page

## Pre-conditions:
• The user is authorized

• At least one product is added to the cart

• The user is on the Checkout page

## Post-conditions:
• The saved address is displayed on the Checkout page

## Steps to Reproduce:
1. Click the "მისამართი" button on the Checkout page - https://www.example.com/
2. Click the button labeled: "ახალი მისამართის დამატება"
3. Fill in at least the required fields
4. Click Save and Continue

## Expected result:
• The website successfully saves the address and details entered by the user
