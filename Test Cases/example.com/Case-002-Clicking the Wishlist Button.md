# Clicking the Wishlist Button

## Description:
Successfully clicking the wishlist button and being redirected to the corresponding page by the user

## Pre-conditions:
• The user is authorized

• At least one product is added to the wishlist

## Post-conditions:
• All products added to the wishlist are displayed

## Steps to Reproduce:
1. Open the website: https://www.example.com/
2. Click the button located on the right side: "სურვილების სია"

## Expected result:
• The button is clicked successfully

• The website redirects the user to the wishlist page
